# Projetos de Desenvolvimento de Jogos

Esta secção contém os meus projetos pessoais relacionados com desenvolvimento de jogos digitais, demonstrando a minha capacidade de conceber, projetar e implementar experiências interativas.

## Visão Geral

Os projetos aqui apresentados foram desenvolvidos por iniciativa própria, durante o meu percurso académico, combinando criatividade com competências técnicas adquiridas em Engenharia Informática.

## Projetos

### [Protótipo de Platformer 2D](https://github.com/goncalo-floro/game)
**Período:** 2025 - Presente
**Estado:** Em desenvolvimento / Documentação

**Descrição:**
Projeto pessoal iniciado a partir de uma ideia original, envolvendo conceção de narrativa, design de níveis e definição de mecânicas base.
