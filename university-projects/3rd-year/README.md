## Projetos

### [Aplicação Móvel - Estádio do Dragão](https://github.com/Estadio-do-Dragao-app)
**UC:** Projeto em Engenharia Informática  
**Período:** 1º Semestre 2025  
**Tecnologias:** Flutter, Dart, Firebase, Google Maps API  
**Equipa:** 5 elementos  
**Nota:** Projeto também listado em projetos técnicos principais

**Descrição:**
Sistema de mapeamento e segurança desenvolvido para utilização em ambiente de estádio, com funcionalidades de localização em tempo real, navegação interior e gestão de emergências.

**Funcionalidades implementadas:**
- Mapeamento interior do estádio com navegação
- Gestão de rotas de evacuação e segurança
- Adaptação a diferentes utilizadores
- Sistema de notificações e alertas

**Competências demonstradas:**
- Desenvolvimento móvel com Flutter
- Trabalho em equipa
- Gestão de requisitos complexos
- Integração de múltiplas APIs e serviços
